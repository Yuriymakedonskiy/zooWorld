Среда



Бесплатный вход в зоопарк по льготным билетам:

  • для детей до 17 лет включительно (от 14 лет и старше – по предъявлению паспорта);
  • для студентов очной формы обучения образовательных учреждений среднего и высшего образования (по предъявлению действительного студенческого билета, либо социальной карты учащегося/студента);
  • для военнослужащих срочной службы и курсантов военных училищ (по предъявлению военного билета);
  • для участников боевых действий (по предъявлению удостоверения участника боевых действий);
  • для инвалидов (по предъявлению пенсионного удостоверения, справки о подтверждении инвалидности либо социальной карты);
  • для пенсионеров (по предъявлению пенсионного удостоверения, паспорта либо социальной карты);
  • для многодетных родителей (по удостоверению многодетных родителей либо паспорту, если там записано три ребёнка и более).


Необходимо предъявить документ, подтверждающий льготу. Льготное посещение действительно только для граждан РФ!
